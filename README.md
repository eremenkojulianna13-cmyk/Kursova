Для початку роботи із застосунком NotaryOfficeApp необхідно підключити базу даних MySQL.
Усі параметри підключення вже налаштовані та не потребують зміни з боку користувача.

Резервна копія бази даних розміщена у папці: NotaryOfficeApp\Database\Database_backup

Імпорт бази даних у MySQL 
1.Для коректної роботи застосунку потрібно виконати імпорт бази даних: 
2.Запустити MySQL Workbench.
3.Відкрити розділ Data Import/Restore.
4.Обрати пункт Import from Self-Contained File. 
5.Вказати шлях до файлу резервної копії бази даних з папки NotaryOfficeApp\Database\Database_backup. 
6.Створити нову схему з назвою notary_office. 
7.Натиснути кнопку Start Import та дочекатися завершення процесу. 
Після виконання цих кроків база даних буде повністю готова до використання.

Параметри підключення до бази даних 
Застосунок використовує фіксовані параметри підключення до MySQL: 
сервер: localhost 
користувач: root 
база даних: notary_office 
пароль: 12345

Дані для входу в застосунок
Пароль для входу в систему: admin123

Дані входу однакові для всіх користувачів і призначені для демонстрації та навчального використання застосунку.
